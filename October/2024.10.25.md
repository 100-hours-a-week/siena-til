#### | Git 동작방식
1. git init 기존에 있는 디렉토리를 Git 저장소로 초기화(.git)폴더 생성
2. git add a.txt하면 가상의 폴더에서 스테이지로 올라감
3. 파일 변경되면 스테이지에서 다시 워킹 디렉토리로 내려옴
4. 다시 깃 에드하면 스테이지로 이동
5. 스테이지에 올려놨다가 커밋하면 히스토리로 올라감
6. 깃푸시하면 서버에 동기화 (즉, 업로드)
