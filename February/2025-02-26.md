    - 2744 대소문자 바꾸기

        처음에 이렇게 풀었다.

        ```python
        string = str(input())
        string_list = list(string)
        reverse_string = ''
        for s in string_list:
        if s.isupper() == True:
            s = s.lower()
            reverse_string += s
        else:
            s = s.upper()
            reverse_string += s
        print(reverse_string)
        ```

        `isupper()` 쓴 걸로 뿌듯해했는데 다른 사람 코드보니

        ```python
        string = input()
        print(string.swapcase())
        ```

        이렇더라ㅠ `swapcase()`는 또 뭔지..
        많이 풀어봐야하는 이유가 여기서 나오는 것 같다..
