2. 스터디 시간에 푼 문제

    - 프로그래머스 Lv1. 숫자 짝꿍
    
    `시간초과 난 코드`

    ```python
        def solution(X, Y):
        answer = ''
        X_list = list(map(int, str(X)))
        Y_list = list(map(int, str(Y)))
        
        common = []
        
        for x in X_list:
            if x in Y_list:
                common.append(x)
                Y_list.remove(x)

        # print(common)
        
        if len(common) == 0 :
            answer = '-1'
        elif all(c == 0 for c in common):
            answer = '0'
        else:
            common.sort(reverse= True)
            for c in common:
                answer += str(c)
        
        return answer
    ```
